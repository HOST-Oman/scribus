Sélection d'objets

deselectAll(...)
deselectAll()

Désélectionne tous les objets du document au complet.

getSelectedObject(...)
getSelectedObject([nr]) -> chaîne

Retourne le nom de l'objet sélectionné. "nr", s'il est fourni, indique le numéro de l'objet sélectionné, par exemple 0 signifie le premier objet sélectionné, 1, le second objet sélectionné et ainsi de suite.

moveSelectionToBack(...)
moveSelectionToBack()

Déplace les objets contenus dans la sélection courante à l'arrière plan.

moveSelectionToFront(...)
moveSelectionToFront()

Déplace les objets contenus dans la sélection courante au premier plan.

selectionCount(...)
selectionCount() -> nombre entier

Retourne le nombre d'objets sélectionnés.

selectObject(...)
selectObject("nom")

Ajoute l'objet nommé "nom" à la sélection courante.

Dans le cas où la sélection courante n'est pas vide et si vous désirez utiliser l'objet "nom" en tant qu'objet sélectionné courant pour des opérations ultérieures, vous devez appeler deselectAll() avant d'appeler cette fonction.
