Примечания по импорту HTML в Scribus

В Scribus имеется модуль импорта HTML, который может импортировать чистый, правильно формированный HTML-код и сохранять большую часть разметки и форматирования, при условии, что форматирование или стиль взято из простой разметки HTML, а не из таблиц стилей CSS. Поддержка CSS будет добавлена в будущем.

Итак, в каком именно виде поддерживается HTML и как всё это работает?

При импорте модуль импорта создаст стили абзацев, соответствующие разметке HTML. Полужирный, курсивный и моноширинный текст и выравнивание также поддерживаются. Ниже приведён список поддерживаемых видов HTML-разметки — как верхний, так и нижний регистр тегов.

  • body, div, a — Текст должен находиться между тегами <body>.
  • p и br — Соответствуют разрыву абзаца и строки.
  • H1-H4 — Соответствует размеру заголовков (от 1 до 4).
  • ol, ul, li — Соответствуют упорядоченным или неупорядоченным спискам.
  • pre и code — Соответствуют предварительно отформатированному тексту и фрагментам исходного кода. Они будут преобразованы в текст при помощи моноширинного шрифта Courier.
  • Сетевые ссылки преобразуются в текст с выделением их синим цветом, аналогично тому, как это происходит в большинстве веб-браузеров по умолчанию.
  • b, u, i, em, strong, sub, sup, del, u — Теги форматирования текста, преобразующие стиль текста соответствующим образом. Обратите внимание на то, что ваш шрифт по умолчанию в Scribus должен содержать все эти варианты стилей.

Итак, приведённый ниже фрагмент HTML-кода создаст стили абзацев в Scribus.

<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<title></title>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
</head>
<body>
<h1>H1 Text</h1>
<h2>H2 Text</h2>
<h3>H3 Text</h3>
<h4>H4 Text</h4>
<ol>
<li>Ordered List Item 1  </li>
<li>Ordered List Item 2  </li>
 </ol>
 <ul>
<li>Un-Ordered List Item 1  </li>
 <li>Un-Ordered List Item 2  </li>
 </ul>
 <code>code listings</code>
 <p><b>Bold Paragraph Style</b></p>
 <p><i>Italic Paragraph Style</i></p>
 <p align="center">Centered Text</p>
</body>
</html>
