Работа со слоями

С понятием слоёв вы можете быть знакомы по опыту работы с приложениями для рисования или изменения изображений. Слои могут использоваться для различных целей, например для разделения печатаемых и непечатаемых элементов, таких как сетки, для создания мультиязычных версий документа в одном файле или просто для облегчения редактирования определённых объектов без затрагивания остальных.

По умолчанию, в новом документе имеется всего один слой с именем «Фон». Вновь добавленные объекты будут находиться каждый на своём уровне на этом слое, так что вы можете иметь много уровней на одном слое. Для большинства документов будет достаточно одного слоя, но иногда будет крайне желательно использовать несколько.

Для добавления нового слоя откройте диалоговое окно слоёв в меню Окна > Слои или нажмите клавишу F6:


На первый взгляд всё кажется просто, но на деле это крайне мощный инструмент. В верхней части окна можно задать режим смешивания и непрозрачность слоя.

В нижней части окна имеются базовые инструменты для управления слоями. Слева направо:

  • Добавление нового слоя
  • Удаление выбранного слоя
  • Дублирование выбранного слоя со всем его содержимым
  • Перемещение выбранного слоя на уровень выше
  • Перемещение выбранного слоя на уровень ниже

Основная часть диалогового окна отведена под список слоёв с их различными параметрами. Эти параметры, слева направо:

  • Цветовой индикатор слоя: каждый слой имеет уникальный цвет, помогающий идентифицировать объекты на этом слое. Включить цветовые индикаторы для объектов можно в меню Вид > Документ > Показывать индикаторы слоёв. Для изменения цвета индикатора просто щёлкните по цвету соответствующего слоя в диалоговом окне. При этом будет открыто окно выбора цвета, где вы можете либо выбрать другой цвет, либо создать новый.
  • Видимость слоя: здесь можно сделать слой видимым или невидимым. Видимость не влияет на печать, поэтому будьте осторожны при использовании этой функции, ведь если вы забудете отключить вывод на печать, скрытое содержимое может стать для вас неприятной неожиданностью при печати или экспорте в PDF.
  • Вывод слоя на печать: этот параметр похож на предыдущий — с ним тоже следует быть осторожным, так как видимый слой может быть не установлен для вывода на печать или экспорт в PDF.
  • Блокирование слоя: данный параметр позволяет заблокировать все объекты на слое, то есть предотвратить их изменение.
  • Обтекание текстом для объектов слоя: если флажок установлен, текст в блоках из нижележащих слоёв будет обтекать объекты на выбранном слое.
  • Каркасный режим для объектов слоя: данный параметр будет полезен для ускорения работы Scribus, поскольку все объекты на слое будут отображаться только в виде «каркаса». Это особенно полезно, когда слой содержит сложную векторную графику.
  • Имя слоя: каждый слой имеет собственное имя, которое здесь и показывается. Для изменения имени слоя дважды щёлкните левой кнопкой мыши по полю «Имя» и введите новое.

Переключение между слоями

Есть два способа переключения между слоями. Первый заключается в выборе слоя в диалоговом окне слоёв, но открывать диалоговое окно каждый раз, когда нужно всего лишь изменить активный слой и ничего более, немного утомительно. К счастью, в строке состояния в нижней части окна Scribus есть поле выбора слоя. В нём в алфавитном порядке перечислены все слои документа, а также указаны их цвета.

Перемещение объектов между слоями

Если вы хотите переместить объект из одного слоя в другой, воспользуйтесь функцией «Перенести в слой» из контекстного меню объекта или из меню Объект. Эта команда также доступна в контекстном меню объектов в структуре документа.

Советы по работе со слоями

  • Слои всегда существуют для всего документа, а не только для конкретной страницы.
  • Для выбора всех объектов в слое используйте функцию Правка > Выделение по критериям из строки меню.
  • Если вы удаляете слой, где присутствуют объекты, появится окно для подтверждения удаления этих объектов. При отказе от их удаления, они будут перемещены в фоновый слой.
  • Слои документа в Scribus не следует путать со слоями в PSD-файлах. Оба понятия полностью независимы.
  • Scribus поддерживает слои в файлах программы Adobe Illustrator (AI), но только в том случае, когда AI-файл открыт (Файл > Открыть). Если же вы импортируете AI-файл, содержащий слои (Файл > Импортировать > Вставить векторный графический файл), все слои AI-документа смешаются в один.
  • Слои также поддерживаются в PDF версии 1.5 и выше, но на текущий момент только Adobe Reader 7 и более поздние версии способны отображать слои в PDF-документах корректно. Scribus может осуществлять экспорт в PDF 1.5 со слоями, но для этого необходимо установить флажок «Включить слои» на вкладке «Общие» диалогового окна экспорта в pdf.
