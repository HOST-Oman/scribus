Справочник по командной строке Scribus

Scribus, как и любую другую программу, можно запустить из командной строки. Здесь приведены параметры командной строки для систем Linux/UNIX (включая Mac OS X). В других операционных системах параметры могут использоваться по-другому.


  • -f, --file
    Открывает указанный файл. Возможно просто передать имя файла в качестве безусловного аргумента, хотя, если имя начинается с тире (-), вам нужно будет использовать --, например, scribus -- -myfile.sla.
  • -fi, --font-info
    Показывает вывод подключения шрифтов при запуске программы. Это может быть использовано для диагностики отсутствующих глифов в шрифтах или потенциально повреждённых файлов шрифтов.
  • -h, --help
    Выводит краткую сводку по использованию.
  • -l, --lang xx
    Переопределяет язык системы по умолчанию и запускает Scribus на языке xx. Язык задается с теми же кодами стандарта POSIX, которые используются в переменных среды LANG и LC_ALL. Например, английский язык может быть выбран при помощи параметров «en» (обобщённый английский), «en_GB» (британский английский), «en_US» (американский английский) и так далее. Аналогично, для выбора изменённого немецкого используйте параметры «de» или «de_DE», для традиционного немецкого — «de_1901», швейцарского немецкого — «de_CH».
  • -la, --langs-available
    Выводит список языков, для которых доступны переводы пользовательского интерфейса. Для использования определённого языка запустите Scribus при помощи команды scribus -l xx, где xx — короткий код этого языка.
  • -nns, --never-splash
    Останавливает отображение заставки при запуске программы. Создаёт пустой файл с именем .neversplash в каталоге ~/.scribus.
  • -ns, --no-splash
    Подавляет отображение заставки во время запуска Scribus.
  • -pi, --profile-info
    Выводит список цветовых профилей, доступных для использования в Scribus. Это может быть использовано для диагностики отсутствующих или повреждённых цветовых профилей.
  • -pr, --prefs <filename>
    Использует имя файла в качестве пути для пользовательских параметров. Дополнительная информация доступна в примечаниях ниже.
  • -py, --python-script <script&gt: [arguments ...]
    Выполняет сценарий на языке Python [с необязательными аргументами]. Этот параметр должен использоваться самым последним.
  • -sb, --swap-buttons
    Устанавливает зеркальное отображение кнопок в диалоговых окнах (например, Отмена/Нет/Да вместо Да/Нет/Отмена).
  • -u, --upgradecheck
    Получает с сервера Scribus информацию с указанием последних доступных версий.
  • -v, --version
    Выводит номер версии Scribus и завершает работу.
Примечания:

В версии 1.5.3 произошли изменения в расположении различных файлов настроек и пользовательских данных.

До этой версии все они хранились в каталоге .scribus, где можно было найти файлы charpalette.ucp, checkfonts.xml, prefs140.xml, scribus140.rc, scribusshapes.xml и scribus-short-words.rc, а также каталоги downloads, palettes, plugins, scrapbook, swatches и temp.

Теперь же файлы checkfonts.xml, prefs150.xml, scribus150.rc и scribusshapes.xml находятся в каталоге .config/scribus.
В .local/share/scribus можно обнаружить файлы charpalette.ucp, scribus-short-words.rc и каталоги cache, doc, palettes, plugins, scrapbook и temp.
