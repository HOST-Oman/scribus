Автоматическая нумерация страниц

Добавление автоматически увеличивающихся номеров страниц не такое простое, как, например, в текстовом процессоре — по крайней мере до тех пор, пока вы его не освоите. Более того, когда дело доходит до смешанных схем нумерации (например, I, II, III для вступления, 1, 2, 3 для основного текста, A, B, C для приложений), то в Scribus всё становитс даже проще, чем в текстовом процессоре.

Как вы уже знаете, почти всё в Scribus должно содержаться в блоке, и номера страниц — не исключение. Таким образом, чтобы вставить номер страницы, необходимо сначала создать текстовый блок. Войдя в режим изменения содержимого блока, мы можем вставить переменный номер страницы, чтобы указать программе на необходимость использования реального номера страницы. Вставить эту переменную можно двумя различными способами. Первый — воспользоваться сочетанием клавиш Ctrl+Alt+Shift+P на клавиатуре. Если вы не хотите сломать себе пальцы, можете переопределить это сочетание в дилоговом окне настройки, либо же просто использовать команду меню Вставка > Символы > Номер страницы. Теперь вы можете видеть, что Scribus показывает номер текущей страницы, но эффект может быть не совсем тем, что ожидалось, поскольку номер страницы добавлен только к текущей выбранной странице. Если вы хотите автоматизировать нумерацию страниц (также называемую «разбивкой на страницы»), следует вставить переменную в текстовый блок на мастер-странице. Как вы заметите, Scribus не отображает реальный номер на мастер-странице, потому что мастер-страница не является реальной страницей документа и поэтому не имеет позиции в последовательности страниц. Вместо этого будет отображаться символ подстановки #.

Изменение схемы нумерации

Для изменения схемы нумерации необходимо воспользоваться меню Файл > Параметры документа. Во вкладке Разделы можно создать разделы нумерации для документа:

Предположим, у вас есть введение, которое занимает первые три страницы, и вы хотите, чтобы эти страницы были пронумерованы римскими цифрами. Основной текст начнётся на пятой физической странице с номером 1. По эстетическим соображениям на странице 4 вообще не должно быть номера. Первый раздел уже создан. Вы можете задать ему имя в столбце «Имя». В столбце «От» установите значение «1», в столбце «До» — значение «3». В качестве стиля нумерации укажите «I, II, III, ...», в столбце «Начало» задайте значение «1».

Чтобы добавить новый раздел, щёлкните кнопку «Добавить» в нижней части диалогового окна. Для четвёртой страницы в нашем примере значения параметров будут следующими: «От» = 4, «До» = 4, «Стиль» и «Начало» здесь не имеют значения, но важно, чтобы был снят флажок во втором столбце «Показывать», потому что именно он отвечает за сокрытие номера страницы. Для основного текста значения такие: «От» = 5, «До» = [количество страниц, необходимое для основной части текста], «Стиль» = 1, 2, 3, «Начало» = 1. Вот и всё, ваш документ будет разбит на страницы так, как вы хотели.

Советы по нумерации страниц

  • Если у вас есть несколько мастер-страниц, вам нужно будет добавить символ подстановки на каждую мастер-страницу, где, как ожидается, должен присутствовать номер страницы.
  • Чтобы номер страницы гарантированно находится в одном и том же месте на каждой мастер-странице, сначала создайте одну такую страницу с подходящей вам разметкой, а затем просто дублируйте её.
  • Если вы работаете с крупными документами, убедитесь, что текстовый блок имеет достаточный размер для того, чтобы в нём поместились все разряды номера страницы. В противном случае произойдёт переполнение текста, и номер страницы останется невидимым.
  • Вы можете применять стили и прямое форматирование текста по своему усмотрению, как и для обычного текстового блока.
  • В европейских изданиях нумерация страниц всегда начинается с правой страницы, то есть страница 1 должна быть правой страницей.
  • Хорошим стилем и признаком качества считается то, что в книгах, написанных на каком-либо европейском языке, новые главы всегда начинаются на правой странице, то есть на странице с нечётным номером.
